# Pre-requisites

1. Docker
2. Docker Compose
3. Docker Image `docker pull bitnami/openldap`
4. Persistent Storage `./data/openldap:/bitnami/openldap`
5. LDAP Client Tools(Optional) like `ldapsearch`, `ldapadd`, `ldapmodify`, `slapcat` and so on..
